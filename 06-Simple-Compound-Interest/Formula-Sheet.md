# Simple & Compound Interest — Formula Sheet

| If question says... | Use this formula/rule | Go to examples |
|---|---|---|
| Find SI | SI = (P×R×T)/100 | [01-Simple-Interest.md](./01-Simple-Interest.md) |
| Find amount in SI | Amount = P + SI | [01-Simple-Interest.md](./01-Simple-Interest.md) |
| Find CI amount | Amount = P(1+R/100)^T | [02-Compound-Interest.md](./02-Compound-Interest.md) |
| Half-yearly/quarterly CI | Replace R and T accordingly | [02-Compound-Interest.md](./02-Compound-Interest.md) |
| Difference between CI and SI | For 2 years: Difference = P(R/100)^2 | [03-CI-vs-SI-Comparison.md](./03-CI-vs-SI-Comparison.md) |

## Compounding Rules

- Half-yearly: R -> R/2, T -> 2T
- Quarterly: R -> R/4, T -> 4T
