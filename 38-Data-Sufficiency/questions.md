# Data Sufficiency — Practice Questions

Use standard DS options:
1) A alone sufficient
2) B alone sufficient
3) Both together sufficient
4) Each alone sufficient
5) Not sufficient even together

1. Is R older than S?
   A: R was born in 2001.
   B: S was born in 2003.

2. Is P the sister of Q?
   A: P and Q have same parents.
   B: P is female.

3. What is the code for CAT?
   A: C=3, A=1, T=20.
   B: Code uses alphabetical positions.

4. Is x > y?
   A: x - y = 5
   B: y = 2

5. What is father’s age?
   A: Father is 3 times son.
   B: Son is 12 years old.
