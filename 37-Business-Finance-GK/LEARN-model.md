# LEARN Model

LEARN is a communication framework often expanded as Listen, Explain, Acknowledge, Recommend, Negotiate.

1. In LEARN, first step is usually:  
   A) Negotiate B) Listen C) Recommend D) Reject  
   **Answer: B**
2. LEARN model is mainly used for:  
   A) Bridge design B) Communication and counselling C) Tax filing D) Encryption  
   **Answer: B**
