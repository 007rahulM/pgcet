# Factorization & Algebraic Expressions — Answers

### Q1
Need sum 9, product 20 → 4 and 5.
```
(x + 4)(x + 5)
```
**Answer:** (x + 4)(x + 5)

---

### Q2
```
(x + 6)(x - 6) = x^2 - 36
```
**Answer:** x^2 - 36

---

### Q3
Common factor 4x:
```
8x^2 + 12x = 4x(2x + 3)
```
**Answer:** 4x(2x + 3)

---

### Q4
Need sum -1, product -12 → -4 and 3.
```
(x - 4)(x + 3)
```
**Answer:** (x - 4)(x + 3)

---

### Q5
```
(3x - 2)^2 = 9x^2 - 12x + 4
```
**Answer:** 9x^2 - 12x + 4

---

### Q6
```
5x^2 - 20 = 5(x^2 - 4) = 5(x - 2)(x + 2)
```
**Answer:** 5(x - 2)(x + 2)
