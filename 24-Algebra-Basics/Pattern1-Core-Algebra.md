## 🎯 MASTER FORMULA SHEET

### 1. BASIC ALGEBRAIC IDENTITIES
```
(a + b)² = a² + 2ab + b²
(a - b)² = a² - 2ab + b²
a² - b² = (a - b)(a + b)
(a + b)³ = a³ + 3a²b + 3ab² + b³
(a - b)³ = a³ - 3a²b + 3ab² - b³
(a + b + c)² = a² + b² + c² + 2(ab + bc + ca)
```

### 2. FACTORIZATION PATTERNS
```
ax + ay = a(x + y)
ax - ay = a(x - y)
ax + bx = x(a + b)
a² + 2ab + b² = (a + b)²
a² - 2ab + b² = (a - b)²
a² - b² = (a - b)(a + b)
```

### 3. LINEAR EQUATIONS
**Single variable:**  
```
ax + b = 0  →  x = -b/a
```

**Two variables (simultaneous):**
```
a1x + b1y = c1
a2x + b2y = c2

x = (c1b2 - c2b1) / (a1b2 - a2b1)
y = (a1c2 - a2c1) / (a1b2 - a2b1)
```

### 4. INDICES (EXPONENT RULES)
```
a^m × a^n = a^(m+n)
a^m / a^n = a^(m-n)
(a^m)^n = a^(mn)
(ab)^n = a^n b^n
a^0 = 1 (a ≠ 0)
a^-n = 1/a^n
```

### 5. LOGARITHMS
```
log_a (xy) = log_a x + log_a y
log_a (x/y) = log_a x - log_a y
log_a (x^n) = n log_a x
log_a a = 1,  log_a 1 = 0
Change of base: log_a b = (log_c b) / (log_c a)
```

### 6. PROGRESSIONS
**Arithmetic Progression (AP):**
```
a_n = a + (n-1)d
S_n = n/2 [2a + (n-1)d]
```

**Geometric Progression (GP):**
```
a_n = ar^(n-1)
S_n = a(r^n - 1)/(r - 1),  r ≠ 1
S_∞ = a/(1 - r),  |r| < 1
```

**Harmonic Progression (HP):**
If 1/a, 1/b, 1/c are in AP → a, b, c are in HP.  
```
b = 2ac / (a + c)
```

### 7. BINOMIAL THEOREM
```
(a + b)^n = Σ [C(n,k) a^(n-k) b^k],  k = 0 to n
```
```
C(n,k) = n! / [k!(n-k)!]
```
**Middle term (n even):** (n/2 + 1)th term  
**Middle terms (n odd):** (n+1)/2 and (n+3)/2 terms

## ✅ SOLVED EXAMPLES (EXAM STYLE)

### Example 1: Expansion
**Expand:** (x + 3)²  
```
(x + 3)² = x² + 6x + 9
```

---

### Example 2: Factorization
**Factorize:** x² - 9  
```
x² - 9 = (x - 3)(x + 3)
```

---

### Example 3: Simultaneous Equations
**Solve:** 2x + 3y = 12,  x - y = 1  
```
From x - y = 1 → x = y + 1
Substitute: 2(y+1) + 3y = 12 → 5y = 10 → y = 2
x = 3
```

---

### Example 4: Indices
**Simplify:** (2^3 × 2^5) / 2^4  
```
2^(3+5-4) = 2^4 = 16
```

---

### Example 5: Logarithms
**Find:** log₁₀(1000)  
```
10^3 = 1000 → log₁₀(1000) = 3
```

---

### Example 6: AP
**Find 10th term:** a = 5, d = 3  
```
a_10 = 5 + 9×3 = 32
```

---

### Example 7: GP
**Sum of first 4 terms:** a = 2, r = 3  
```
S_4 = 2(3^4 - 1)/(3 - 1) = 2(81 - 1)/2 = 80
```

---

### Example 8: Binomial Theorem
**Coefficient of x² in (x + 2)^4**  
```
Term with x²: C(4,2) x² 2^2 = 6 × 4 = 24
```
