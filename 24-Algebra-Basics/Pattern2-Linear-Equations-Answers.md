# Linear & Simultaneous Equations — Answers

### Q1
5x + 9 = 24
```
5x = 15
x = 3
```
**Answer:** 3

### Q2
3x + 2y = 16, x + y = 6
From x + y = 6, x = 6 - y
```
3(6 - y) + 2y = 16
18 - 3y + 2y = 16
-y = -2
```
So y = 2, x = 4.

**Answer:** x = 4, y = 2

---

### Q3
4x - y = 11 and 2x + y = 7
Add the equations:
```
6x = 18
x = 3
```
Then 2x + y = 7 gives y = 1.

**Answer:** x = 3, y = 1
