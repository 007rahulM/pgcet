# Algebra Basics — Overview

This folder now follows the same **learn → pattern → practice → answers** flow as the stronger folders in the repo.

Algebra basics are a high-value foundation topic because they support **Quadratic Equations, Coordinate Geometry, Trigonometry, and even some Data Interpretation and Logical sections**.

## Golden Rules

- **Write the formula first, then substitute.**
- **Look for a pattern before expanding blindly.**
- **If a problem looks long, try factorization or substitution first.**
- **For AP/GP/binomial, identify the pattern word in the question immediately.**

## Study Order

1. Start with **Pattern 1** for the full picture.
2. Master **Pattern 5** and **Pattern 2** next — these are used most often.
3. Then do **Pattern 3** and **Pattern 4** for formula-based questions.
