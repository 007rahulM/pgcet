# Indices & Logarithms — Pattern 3

## 🧠 Core Idea

Use **exponent rules** and **log laws** to simplify expressions quickly.

---

## ✅ Indices (Exponent Rules)
```
a^m × a^n = a^(m+n)

a^m / a^n = a^(m-n)

(a^m)^n = a^(mn)

(ab)^n = a^n b^n

a^0 = 1 (a ≠ 0)

a^-n = 1/a^n
```

---

## ✅ Logarithm Laws
```
log_a(xy) = log_a x + log_a y

log_a(x/y) = log_a x - log_a y

log_a(x^n) = n log_a x

log_a a = 1,  log_a 1 = 0

Change of base: log_a b = (log_c b)/(log_c a)
```

---

## 🔥 Example 1 (Indices)

**❓ Question:** Simplify (2^3 × 2^5) / 2^4

**✏️ My Solution:**
```
2^(3+5-4) = 2^4 = 16
```
**✅ Answer: 16**

---

## 🔥 Example 2 (Log Laws)

**❓ Question:** Simplify log10(200) − log10(2)

**✏️ My Solution:**
```
log10(200/2) = log10(100) = 2
```
**✅ Answer: 2**

---

## 🔥 Example 3 (Solve for x)

**❓ Question:** If 3^x = 81, find x.

**✏️ My Solution:**
```
81 = 3^4 → x = 4
```
**✅ Answer: 4**

---

## ⚡ Shortcuts

- If numbers match base, **convert to powers** first.
- For log equations, **combine logs** and compare powers.
