# Pattern 3: Indices & Logarithms

## 🔍 How to Recognize This Pattern

- "Simplify powers"
- "Find x if 2^(x+1) = 64"
- "Evaluate log"
- "Use change of base"

---

## 🧠 The Golden Rule

> **First make the bases look the same. Then use exponent or log laws.**

## 📐 Core Formulas

### Indices
```
a^m × a^n = a^(m+n)
a^m / a^n = a^(m-n)
(a^m)^n = a^(mn)
a^0 = 1
a^-n = 1/a^n
```

### Logarithms
```
log_a(xy) = log_a x + log_a y
log_a(x/y) = log_a x - log_a y
log_a(x^n) = n log_a x
log_a a = 1
log_a 1 = 0
log_a b = (log_c b)/(log_c a)
```

---

## ✅ Step-by-Step Examples

### Example 1
**❓ Question:** Simplify (3^5 × 3^-2) / 3^1.

**✏️ My Solution:**
```
3^(5-2-1) = 3^2 = 9
```

**✅ Answer: 9**

---

### Example 2
**❓ Question:** Find x if 2^(x+2) = 32.

**🤔 What I understood:** Convert 32 into a power of 2.

**✏️ My Solution:**
```
32 = 2^5
2^(x+2) = 2^5
x + 2 = 5
x = 3
```

**✅ Answer: 3**

---

### Example 3
**❓ Question:** Simplify log10(500) - log10(5).

**✏️ My Solution:**
```
log10(500/5) = log10(100) = 2
```

**✅ Answer: 2**

---

### Example 4
**❓ Question:** Find log2(32).

**✏️ My Solution:**
Since 2^5 = 32,
```
log2(32) = 5
```

**✅ Answer: 5**

---

## ⚡ 60-Second Strategy

1. Rewrite numbers in the same base if possible.
2. Combine exponents before calculating.
3. For logs, join or split terms using laws.
4. If stuck, change the logarithm into exponential form.
