# Progressions & Binomial Theorem — Pattern 4

## 🧠 Core Idea

Memorize **AP/GP formulas** and use **binomial coefficients** for expansion questions.

---

## ✅ Arithmetic Progression (AP)
```
a_n = a + (n-1)d

S_n = n/2 [2a + (n-1)d]
```

## ✅ Geometric Progression (GP)
```
a_n = ar^(n-1)

S_n = a(r^n - 1)/(r - 1)   (r ≠ 1)

S_∞ = a/(1 - r)  (|r| < 1)
```

## ✅ Harmonic Progression (HP)
If 1/a, 1/b, 1/c are in AP → a, b, c are in HP.
```
b = 2ac/(a + c)
```

---

## ✅ Binomial Theorem
```
(a + b)^n = Σ [C(n,k) a^(n-k) b^k]
```
```
C(n,k) = n! / [k!(n-k)!]
```
**Middle term:**
- n even → (n/2 + 1)th term
- n odd → (n+1)/2 and (n+3)/2 terms

---

## 🔥 Example 1 (AP)

**❓ Question:** a = 4, d = 3. Find 10th term.

**✏️ My Solution:**
```
a_10 = 4 + 9×3 = 31
```
**✅ Answer: 31**

## 🔥 Example 2 (GP)

**❓ Question:** a = 2, r = 3. Find S_4.

**✏️ My Solution:**
```
S_4 = 2(3^4 - 1)/(3 - 1) = 2(81 - 1)/2 = 80
```
**✅ Answer: 80**

---

## 🔥 Example 3 (Binomial)

**❓ Question:** Coefficient of x^2 in (x + 2)^4

**✏️ My Solution:**
```
Term = C(4,2) x^2 2^2 = 6 × 4 = 24
```
**✅ Answer: 24**
