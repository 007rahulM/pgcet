## Quick Reference

### 2×2 Determinant:
> **|A| = ad − bc** for A = [[a,b],[c,d]]

### 2×2 Inverse:
> A⁻¹ = (1/|A|) × [[d,−b],[−c,a]]

### Matrix Multiplication:
- m×p matrix × p×n matrix = m×n matrix
- Inner dimensions must match!

### Key Properties:
- |AB| = |A| × |B|
- Any matrix × Identity = same matrix
- If |A| = 0 → matrix is singular (no inverse)
