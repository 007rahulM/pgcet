# Pattern 4: Conics & Transformations

## 🔍 How to Recognize This Pattern

- "Find focus/directrix"
- "Identify parabola / ellipse / hyperbola"
- "Use standard form"
- "Translate / rotate / scale a point"

---

## 🧠 The Golden Rule

> **Match the given equation to a standard conic form before doing anything else.**

## 📐 Standard Forms

### Parabola
```
y^2 = 4ax   → Focus (a, 0), Directrix x = -a
x^2 = 4ay   → Focus (0, a), Directrix y = -a
```

### Ellipse
```
x^2/a^2 + y^2/b^2 = 1,  a > b
c^2 = a^2 - b^2
Foci = (±c, 0)
```

### Hyperbola
```
x^2/a^2 - y^2/b^2 = 1
c^2 = a^2 + b^2
Foci = (±c, 0)
```

### Transformations
```
Translation: (x, y) → (x + h, y + k)
Rotation: x' = xcosθ - ysinθ,  y' = xsinθ + ycosθ
Scaling: (x, y) → (kx, ky)
```

---

## ✅ Step-by-Step Examples

### Example 1
**❓ Question:** For y^2 = 12x, find focus and directrix.

**✏️ My Solution:**
Compare with y^2 = 4ax.
```
4a = 12 → a = 3
```
Focus = (3, 0), Directrix = x = -3.

**✅ Answer: Focus (3, 0), Directrix x = -3**

---

### Example 2
**❓ Question:** Find foci of x^2/25 + y^2/9 = 1.

**✏️ My Solution:**
```
a^2 = 25, b^2 = 9
c^2 = 25 - 9 = 16
c = 4
```
Foci = (±4, 0).

**✅ Answer: (4, 0) and (-4, 0)**

---

### Example 3
**❓ Question:** Identify the conic x^2/9 - y^2/4 = 1.

**✏️ My Solution:**
There is one plus and one minus term.
That is the standard form of a hyperbola.

**✅ Answer: Hyperbola**

---

## ⚡ 60-Second Strategy

1. Check the signs: same sign → circle/ellipse; opposite signs → hyperbola.
2. For parabola, look for only one squared variable.
3. Compare the coefficient with 4a.
4. For transformations, just apply the rule directly to the coordinates.
