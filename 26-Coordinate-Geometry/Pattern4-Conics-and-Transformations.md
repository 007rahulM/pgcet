# Conics & Transformations — Pattern 4

## 🔍 When This Comes Up
- "Focus/directrix" for parabola, ellipse, or hyperbola
- Identify conic from equation form
- Translation/rotation/scaling questions

---

## 🧠 Core Idea
Recognize **standard conic forms**, then read key parameters directly.

---

## 📐 Key Forms
Parabola:
```
y^2 = 4ax   → Focus (a, 0), Directrix x = -a
x^2 = 4ay   → Focus (0, a), Directrix y = -a
```
Ellipse:
```
(x^2/a^2) + (y^2/b^2) = 1  (a > b)
Foci: (±c, 0), c^2 = a^2 - b^2
```
Hyperbola:
```
(x^2/a^2) - (y^2/b^2) = 1
Foci: (±c, 0), c^2 = a^2 + b^2
```
Transformations:
```
Translation: (x, y) → (x + h, y + k)
Rotation (θ): x' = x cosθ - y sinθ, y' = x sinθ + y cosθ
Scaling: (x, y) → (kx, ky)
```

---

## ✅ Worked Example

**❓ Question:** Find foci of x^2/25 + y^2/9 = 1.

**✏️ My Solution:**
```
a^2 = 25, b^2 = 9
c^2 = a^2 - b^2 = 16 → c = 4
Foci: (±4, 0)
```
**✅ Answer:** (4, 0) and (−4, 0)

---

## 📝 Practice
1. For y^2 = 12x, find focus and directrix
2. Identify conic: x^2/9 - y^2/4 = 1
3. Translate (x, y) by (3, −2)

**Answers:**
1) Focus (3, 0), Directrix x = −3
2) Hyperbola
3) (x + 3, y − 2)
