# Pattern 3: Circles

## 🔍 How to Recognize This Pattern

- "Find center and radius"
- "Write the equation of circle"
- "Convert from general form to standard form"

---

## 🧠 The Golden Rule

> **Compare the equation with standard form first. If it is in general form, read g and f carefully.**

## 📐 Core Forms

**Standard form**
```
(x - h)^2 + (y - k)^2 = r^2
```
Center = (h, k), Radius = r

**General form**
```
x^2 + y^2 + 2gx + 2fy + c = 0
```
Center = (-g, -f)
Radius = √(g^2 + f^2 - c)

---

## ✅ Step-by-Step Examples

### Example 1
**❓ Question:** Find center and radius of x^2 + y^2 - 6x + 8y - 11 = 0.

**✏️ My Solution:**
```
2g = -6 → g = -3
2f = 8  → f = 4
c = -11
Center = (-g, -f) = (3, -4)
Radius = √(9 + 16 + 11) = √36 = 6
```

**✅ Answer: Center (3, -4), Radius 6**

---

### Example 2
**❓ Question:** Write the equation of circle with center (2, -1) and radius 5.

**✏️ My Solution:**
Use standard form:
```
(x - 2)^2 + (y + 1)^2 = 25
```

**✅ Answer: (x - 2)^2 + (y + 1)^2 = 25**

---

### Example 3
**❓ Question:** Find the radius of x^2 + y^2 + 4x - 10y + 13 = 0.

**✏️ My Solution:**
```
2g = 4 → g = 2
2f = -10 → f = -5
c = 13
Radius = √(2^2 + (-5)^2 - 13) = √(4 + 25 - 13) = √16 = 4
```

**✅ Answer: 4**

---

## ⚡ 60-Second Strategy

1. Check whether the equation is standard or general.
2. In general form, divide x and y coefficients by 2 first.
3. Use opposite signs for the center.
4. Find radius only after g, f, c are correct.
