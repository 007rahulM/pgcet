# Circles — Pattern 3

## 🔍 When This Comes Up
- "Find center/radius" from equation
- "Equation of circle" with center and radius given
- Tangent or distance from center questions

---

## 🧠 Core Idea
Convert between **standard** and **general** circle forms to read center/radius quickly.

---

## 📐 Key Formulas
Standard form:
```
(x - h)^2 + (y - k)^2 = r^2
```
General form:
```
x^2 + y^2 + 2gx + 2fy + c = 0
Center = (-g, -f)
Radius = √(g^2 + f^2 - c)
```

---

## ✅ Worked Example

**❓ Question:** Find center and radius of x^2 + y^2 - 6x + 8y - 11 = 0.

**✏️ My Solution:**
```
2g = -6 → g = -3
2f = 8 → f = 4
c = -11
Center = (3, -4)
Radius = √(9 + 16 - (-11)) = √36 = 6
```
**✅ Answer:** Center (3, −4), Radius 6

---

## 📝 Practice
1. Write equation of circle with center (2, -1) and radius 5
2. Find radius of x^2 + y^2 + 4x - 10y + 13 = 0

**Answers:**
1) (x − 2)^2 + (y + 1)^2 = 25
2) r = 4
