# Coordinate Geometry — Overview

This folder now follows the same detailed format as the stronger folders: **recognition clues, formulas, solved examples, practice, and separate answer files**.

Coordinate Geometry becomes easy once you reduce every question to one of a few repeated tools: **distance, midpoint, slope, line equation, circle form, or conic form**.

## Golden Rules

- Plot mentally or draw a tiny sketch.
- Use **difference in x** and **difference in y** first.
- For line questions, slope is the key.
- For circle and conic questions, compare with the standard form before doing anything else.

## Study Order

1. Pattern 2 first — it covers the most-used tools.
2. Pattern 3 next — circle questions are usually direct.
3. Pattern 4 after that — memorize standard forms and focus/directrix facts.
