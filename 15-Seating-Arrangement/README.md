# Seating Arrangement — Overview

Seating arrangement tests your spatial reasoning. Expect **2–4 questions** in MCA PGCET.

## The Golden Rule

> **ALWAYS draw the arrangement! Never solve in your head.**

## Key Concepts

### Linear:
- Fix endpoints first
- "Immediate left/right" = adjacent seat
- Build chains from given order clues

### Circular:
- Fix **one person** at a position to eliminate rotation ambiguity
- "Opposite" = n/2 seats away in n-person circle
- Facing center: Right = clockwise, Left = counterclockwise
