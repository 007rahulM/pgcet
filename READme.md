## 🗂️ How to Use This Repository

1. Open the subject folder you want to study.
2. Read each pattern/subtopic file in order (start from Pattern 1, move forward).
3. After reading, **solve the practice problems** at the bottom of each file.
4. Aim to solve each problem in **under 60 seconds**.

---

## ⏱️ Exam Strategy

- The MCA PGCET exam has **100 questions in 90 minutes** → roughly **54 seconds per question**.
- Focus on speed AND accuracy.
- For Time & Work, Profit & Loss, Percentages — use the LCM method and formula shortcuts.
- For logical sections (Blood Relations, Coding-Decoding, Direction Sense) — draw diagrams.
