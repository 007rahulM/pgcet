
## ЁЯУР FORMULA BLOCK 6 тАФ Combining Ratios

### Question looks like:
- "A:B = 2:3, B:C = 4:5. Find **A:B:C**."
- "A:B = 3:4, B:C = 5:6. Find **A:C**."

### Approach тАФ Make B common:

| Step | What to do |
|------|-----------|
| 1 | Note B's value in both ratios |
| 2 | Find LCM of B's two values |
| 3 | Scale both ratios so B = LCM in both |
| 4 | Read off combined ratio |

### Example:
- A:B = 2:3, B:C = 4:5
- B is 3 in first, 4 in second тЖТ LCM = 12
- Scale first by 4: A:B = 8:12; Scale second by 3: B:C = 12:15
- A:B:C = **8:12:15**
