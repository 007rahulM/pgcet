# Random Variables — Basics

## 🔍 Why This Comes Up
Syllabus includes **random variables**. Expect short formula-based questions.

---

## 📐 KEY DEFINITIONS

**Random Variable (X):**  
A function that assigns a numerical value to each outcome.

**Discrete RV:** Takes countable values (0,1,2...).  
**Continuous RV:** Takes any value in an interval.

---

## 📐 CORE FORMULAS (DISCRETE)

**Expected Value (Mean):**
```
E(X) = Σ x · P(X = x)
```

**Variance:**
```
Var(X) = Σ (x − μ)² P(X = x)
```

**Standard Deviation:**
```
SD = √Var(X)
```

---

## ✅ STANDARD RESULTS

**Bernoulli (0/1):**
```
P(X=1)=p, P(X=0)=q
Mean = p
Variance = pq
```

**Binomial (n, p):**
```
Mean = np
Variance = npq
```

---

## 🔥 SOLVED EXAMPLE

If X takes values 0, 1, 2 with probabilities 0.2, 0.5, 0.3:
```
E(X) = 0×0.2 + 1×0.5 + 2×0.3 = 1.1
```

---

## 📝 PRACTICE (WITH ANSWERS)

1. A fair coin: X=1 for head, 0 for tail. Find E(X).  
2. If p = 0.7 in Bernoulli, find variance.  
3. Binomial with n=5, p=0.2. Find mean.  

**Answers:**  
1) 0.5  
2) pq = 0.21  
3) np = 1
