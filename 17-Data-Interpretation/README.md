# Data Interpretation — Overview

Data Interpretation (DI) tests your ability to read and analyze presented data. Expect **3–5 questions** in MCA PGCET.

## Patterns in This Folder

| File | Pattern | Recognition Clue |
|------|---------|-----------------|
| [Pattern1-Tables-and-Bar-Graphs.md](./Pattern1-Tables-and-Bar-Graphs.md) | Rows/columns of data; vertical/horizontal bars | Table with rows and columns or bar chart |
| [Pattern2-Pie-Charts-and-Line-Graphs.md](./Pattern2-Pie-Charts-and-Line-Graphs.md) | Circular charts; trend lines | Circular diagram with % sectors; points on a grid |
| [Pattern3-Caselets.md](./Pattern3-Caselets.md) | Data given as a paragraph | "A company employs 500 people. 60% are male..." |

## Key Formulas

| Question Type | Formula |
|--------------|---------|
| % change | (New − Old)/Old × 100 |
| % of total | (Part/Total) × 100 |
| Pie chart angle | (Percentage/100) × 360° |
| Average | Sum / Count |

## General Strategy

1. Read the title and units first
2. Read the QUESTION before the data
3. Use approximation for MCQs
4. For caselets: extract all numbers before solving
