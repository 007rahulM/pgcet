## The 60-Second Strategy

**For any series:**
1. Find differences → constant? → Arithmetic
2. Find ratios → constant? → Geometric
3. Are terms squares (n²) or cubes (n³)?
4. Do differences themselves form a pattern? → Second differences
5. Letters → convert to numbers (A=1...Z=26), find the pattern
