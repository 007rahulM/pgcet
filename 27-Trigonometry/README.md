This folder now follows the same detailed folder style: **recognition clues, formulas, solved examples, practice, and separate answer files**.

Trigonometry becomes easy when you split questions into repeating patterns: **standard values, identities, compound angles, triangle rules, inverse trig, and heights & distances**.

## Golden Rules

- Memorize the standard value table once.
- Decide which ratio or identity the question is asking for.
- Draw a small right triangle for heights-and-distances problems.
- For inverse trig, always respect the principal value range.
