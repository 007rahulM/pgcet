### 2. FUNDAMENTAL IDENTITIES

**Pythagorean Identities:**
```
sinВІОё + cosВІОё = 1
1 + tanВІОё = secВІОё
1 + cotВІОё = cscВІОё
```

**Reciprocal Identities:**
```
csc Оё = 1/sin Оё
sec Оё = 1/cos Оё
cot Оё = 1/tan Оё
```

**Quotient Identities:**
```
tan Оё = sin Оё/cos Оё
cot Оё = cos Оё/sin Оё
```

### 3. COMPOUND ANGLE FORMULAS

**Sum and Difference:**
```
sin(A + B) = sin A cos B + cos A sin B
sin(A - B) = sin A cos B - cos A sin B

cos(A + B) = cos A cos B - sin A sin B
cos(A - B) = cos A cos B + sin A sin B

tan(A + B) = (tan A + tan B)/(1 - tan A tan B)
tan(A - B) = (tan A - tan B)/(1 + tan A tan B)
```

**60-Second Trick:**
- Sin: Same (SC + CS)
- Cos: Change sign (CC - SS)

---

### 4. DOUBLE ANGLE FORMULAS

```
sin 2A = 2 sin A cos A
cos 2A = cosВІA - sinВІA = 2cosВІA - 1 = 1 - 2sinВІA
tan 2A = 2tan A/(1 - tanВІA)
```

### 8. SPECIAL ANGLE CONVERSIONS

**Negative Angles:**
```
sin(-Оё) = -sin Оё
cos(-Оё) = cos Оё
tan(-Оё) = -tan Оё
```

**Complementary Angles (ПҖ/2 - Оё):**
```
sin(ПҖ/2 - Оё) = cos Оё
cos(ПҖ/2 - Оё) = sin Оё
tan(ПҖ/2 - Оё) = cot Оё
```

**Supplementary Angles (ПҖ - Оё):**
```
sin(ПҖ - Оё) = sin Оё
cos(ПҖ - Оё) = -cos Оё
tan(ПҖ - Оё) = -tan Оё
```

### PROBLEM 3: sinвҒ»В№(sin Оё) Type

**Question:** Find sinвҒ»В№(sin 3ПҖ/5).

**Solution:**
```
3ПҖ/5 is NOT in range [-ПҖ/2, ПҖ/2]
3ПҖ/5 = ПҖ - 2ПҖ/5 (supplementary angle)
sin(3ПҖ/5) = sin(ПҖ - 2ПҖ/5) = sin(2ПҖ/5)
2ПҖ/5 IS in range [-ПҖ/2, ПҖ/2]
```

**Answer: 2ПҖ/5**

---

### PROBLEM 4: Expression Simplification

**Question:** Find value of 3sin(ПҖ/6)sec(ПҖ/3) - 4sin(5ПҖ/6)cot(ПҖ/4).

**Solution:**
```
Step 1: Find each value
sin(ПҖ/6) = 1/2
sec(ПҖ/3) = 1/cos(ПҖ/3) = 1/(1/2) = 2
sin(5ПҖ/6) = sin(ПҖ - ПҖ/6) = sin(ПҖ/6) = 1/2
cot(ПҖ/4) = 1/tan(ПҖ/4) = 1/1 = 1

Step 2: Calculate
3 Г— (1/2) Г— 2 = 3
4 Г— (1/2) Г— 1 = 2

Step 3: Final answer
3 - 2 = 1
```

**Answer: 1**

---

### PROBLEM 5: Compound Angle

**Question:** Find sin(ПҖ/4 + ПҖ/6).

**Solution:**
```
Using: sin(A + B) = sin A cos B + cos A sin B

sin(ПҖ/4 + ПҖ/6) = sin(ПҖ/4)cos(ПҖ/6) + cos(ПҖ/4)sin(ПҖ/6)
                = (1/вҲҡ2)(вҲҡ3/2) + (1/вҲҡ2)(1/2)
                = вҲҡ3/(2вҲҡ2) + 1/(2вҲҡ2)
                = (вҲҡ3 + 1)/(2вҲҡ2)
```

**Answer: (вҲҡ3 + 1)/(2вҲҡ2)**

### PROBLEM 6: tanвҒ»В№(1)

### PROBLEM 7: Identity Application

**Question:** If sin Оё = 3/5, find cos Оё.

**Solution:**
```
Using: sinВІОё + cosВІОё = 1
(3/5)ВІ + cosВІОё = 1
9/25 + cosВІОё = 1
cosВІОё = 1 - 9/25 = 16/25
cos Оё = Вұ4/5

If Оё in first quadrant, cos Оё = 4/5
```

**Answer: 4/5 (assuming first quadrant)**

### PROBLEM 8: Degree to Radian

**Question:** Convert 120В° to radians.

**Solution:**
```
Radians = Degrees Г— (ПҖ/180)
        = 120 Г— (ПҖ/180)
        = 2ПҖ/3
```

**Answer: 2ПҖ/3**

### PROBLEM 10: Double Angle

**Question:** If sin Оё = 1/2, find sin 2Оё.

**Solution:**
```
Using: sin 2Оё = 2 sin Оё cos Оё

First find cos Оё:
sinВІОё + cosВІОё = 1
(1/2)ВІ + cosВІОё = 1
cosВІОё = 3/4
cos Оё = вҲҡ3/2

Now: sin 2Оё = 2 Г— (1/2) Г— (вҲҡ3/2) = вҲҡ3/2
```

**Answer: вҲҡ3/2**
