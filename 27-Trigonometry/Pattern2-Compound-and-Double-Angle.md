# Pattern 2: Compound & Double Angle Formulas

## 🔍 How to Recognize This Pattern

- "Find sin(A+B)"
- "Find cos(A-B)"
- "Find sin2A / cos2A / tan2A"
- "Simplify trig expression using angle formulas"

---

## 🧠 The Golden Rule

> **Do not expand randomly. First identify whether it is a sum/difference angle or a double-angle question.**

## 📐 Core Formulas

```
sin(A + B) = sinA cosB + cosA sinB
sin(A - B) = sinA cosB - cosA sinB

cos(A + B) = cosA cosB - sinA sinB
cos(A - B) = cosA cosB + sinA sinB

tan(A + B) = (tanA + tanB)/(1 - tanA tanB)
tan(A - B) = (tanA - tanB)/(1 + tanA tanB)
```

Double angle:
```
sin 2A = 2sinA cosA
cos 2A = cos^2A - sin^2A = 2cos^2A - 1 = 1 - 2sin^2A
tan 2A = 2tanA/(1 - tan^2A)
```

## ✅ Step-by-Step Examples

### Example 1
**❓ Question:** Find sin(45° + 30°).

**✏️ My Solution:**
```
sin(45+30) = sin45 cos30 + cos45 sin30
           = (1/√2)(√3/2) + (1/√2)(1/2)
           = (√3 + 1)/(2√2)
```

**✅ Answer: (√3 + 1)/(2√2)**

---

### Example 2
**❓ Question:** If sinA = 3/5 and A is acute, find cos2A.

**✏️ My Solution:**
Since A is acute,
```
cosA = 4/5
cos2A = cos^2A - sin^2A = 16/25 - 9/25 = 7/25
```

**✅ Answer: 7/25**

---

### Example 3
**❓ Question:** Find tan(45° - 30°).

**✏️ My Solution:**
```
tan(45-30) = (tan45 - tan30)/(1 + tan45 tan30)
           = (1 - 1/√3)/(1 + 1/√3)
           = 2 - √3
```

**✅ Answer: 2 - √3**

---

## ⚡ 60-Second Strategy

1. Mark whether the question is about +, -, or 2A.
2. Write the exact identity first.
3. Substitute standard values or given ratios.
4. Simplify at the end.
