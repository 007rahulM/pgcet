### Q2
If cosA = 1/2 and A is acute, sinA = √3/2.
```
sin2A = 2sinA cosA = 2(√3/2)(1/2) = √3/2
```
**Answer:** √3/2

### Q4
```
tan(45+30) = (1 + 1/√3)/(1 - 1/√3) = 2 + √3
```
