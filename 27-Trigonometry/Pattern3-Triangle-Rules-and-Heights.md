# Triangles, Heights & Area — Pattern 3

## 🧠 Core Idea

Use **sine rule**, **cosine rule**, and **area formula** to solve triangle questions fast.

---

## ✅ Sine Rule
```
a/sin A = b/sin B = c/sin C
```

## ✅ Cosine Rule
```
c² = a² + b² - 2ab cos C
```

## ✅ Area Formula
```
Area = (1/2)ab sin C
```

---

## 📌 Triangle Diagram
```
       B
      / \
     /   \
   c/  A  \ b
   /       \
  /_________\
 A     a     C
```

---

## 🔥 Example 1 (Sine Rule)

**❓ Question:** In △ABC, a = 6, A = 30°, B = 45°. Find b.

**✏️ My Solution:**
```
a/sinA = b/sinB
6/sin30 = b/sin45
6/(1/2) = b/(1/√2)
12 = b√2
b = 12/√2 = 6√2
```
**✅ Answer: 6√2**

---

## 🔥 Example 2 (Cosine Rule)

**❓ Question:** a = 5, b = 7, C = 60°. Find c.

**✏️ My Solution:**
```
c² = 5² + 7² - 2(5)(7)cos60
   = 25 + 49 - 70(1/2)
   = 74 - 35 = 39
```
**✅ Answer: c = √39**
