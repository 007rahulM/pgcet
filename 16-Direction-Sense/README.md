# Direction Sense — Overview

Direction sense tests spatial reasoning with maps and compass directions. Expect **2–3 questions** in MCA PGCET.

## The Golden Rule

> **ALWAYS draw a diagram! Track each step on paper.**

## Key Formulas

- **Shortest distance:** √(horizontal² + vertical²)
- **North = +Y, South = −Y, East = +X, West = −X**

## Turns Quick Reference

| Facing | Turn Right → | Turn Left → |
|--------|-------------|------------|
| North | East | West |
| East | South | North |
| South | West | East |
| West | North | South |
