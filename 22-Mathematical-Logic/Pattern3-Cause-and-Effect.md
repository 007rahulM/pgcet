# Pattern 3: Cause and Effect (Statement Logic)

## 🔍 How to Recognize This Pattern

- Two statements A and B are given.
- Options ask: A causes B / B causes A / independent / common cause.

---

## 🧠 Evaluation Rule

1. Check **time order** first (cause happens before effect).
2. Check **logical possibility** (can B naturally follow from A?).
3. Reject options needing extra assumptions not given in statements.

---

## ✅ Step-by-Step Examples

### Example 1
**A:** Heavy rain overnight.  
**B:** Roads were waterlogged in morning.

- Rain happens before waterlogging.
- Direct logical relation exists.

**✅ Answer:** A is the cause, B is the effect.

### Example 2
**A:** City traffic was blocked.  
**B:** A political rally was held in city center.

- Rally can cause traffic blockage.
- Time order and real-world relation are valid.

**✅ Answer:** B is the cause, A is the effect.

### Example 3
**A:** A truck overturned at night.  
**B:** Police cordoned off the area in morning.

- Overturning event happens first.
- Cordon is a response/effect.

**✅ Answer:** A cause, B effect.

---

## ⚡ 60-Second Strategy

- If one statement is an event and the other is an official response, event is usually cause.
- If both statements are only outcomes, select common cause/effects option.
- If both happen independently without relation, select independent causes.

---

## 📝 Practice Problems

1. A: Factory machine failed. B: Production dropped next day.
2. A: Exam date postponed. B: Students got extra preparation time.
3. A: Flood warning issued. B: People moved to shelters.
