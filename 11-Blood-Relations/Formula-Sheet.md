# Blood Relations — Complete Rule Sheet

## 📐 RULE BLOCK 1 — Pointing to a Photo

### Question looks like:
- "Pointing to a man, A says **'His mother is the wife of my father's son.'**"
- "Pointing to a photo: **'She is the daughter of the only son of my grandfather.'**"
- "**'This man's son is my son's uncle.'** How are they related?"

### Step-by-Step Method:

| Step | What to do |
|------|-----------|
| 1 | Find the subject (the person being pointed at or described) |
| 2 | Work from the most distant relative back toward the speaker |
| 3 | Map each relationship word to the correct person |
| 4 | Draw arrows and boxes; male = square, female = circle |

### KEY PHRASES TO DECODE:

| Phrase | Meaning |
|--------|---------|
| "Only son of my grandfather" | My father (if grandfather has one son) |
| "My father's son" | Myself (if only child) or my brother |
| "My mother's only son" | Myself |
| "Son's uncle" | Another son of my father = my brother |
| "Wife of my father's son" | My own wife (if I am the father's son) |
| "Daughter of X's sister" | X's niece |
| "X's father's brother" | X's uncle (paternal) |
| "X's mother's brother" | X's maternal uncle |

## 📐 RULE BLOCK 2 — Chain of Relations (Family Tree)

### Question looks like:
- "A is the father of B. C is the sister of B. D is the husband of A's mother. How is D related to C?"
- "P is Q's brother. R is P's mother. S is R's father. How is S related to Q?"

### Step-by-Step Method:

| Step | What to do |
|------|-----------|
| 1 | Write all persons A, B, C, D as nodes |
| 2 | Connect with labeled arrows showing relation |
| 3 | Note gender where specified (brother/sister, husband/wife) |
| 4 | Trace the path from one person to the other |

→ **See examples: [Pattern2-Chain-and-Family-Tree.md](./Pattern2-Chain-and-Family-Tree.md)**

## 📐 RULE BLOCK 3 — Symbol Coded Relations

### Question looks like:
- "P $ Q means P is the father of Q. M # N means M is the sister of N. Then A $ B # C means?"
- "Decode the given symbol chain to find the relationship."

### Step-by-Step Method:

| Step | What to do |
|------|-----------|
| 1 | Write out the symbol → relation translation table |
| 2 | Replace each symbol with its meaning |
| 3 | Build the family tree step by step |
| 4 | Trace the relationship |

→ **See examples: [Pattern3-Coded-Relations.md](./Pattern3-Coded-Relations.md)**

## 📐 REFERENCE — All Standard Relationships

| Relationship | Meaning |
|-------------|---------|
| Father's or mother's father | Grandfather |
| Father's or mother's mother | Grandmother |
| Father's brother | Paternal uncle |
| Mother's brother | Maternal uncle |
| Father's sister | Paternal aunt |
| Mother's sister | Maternal aunt |
| Uncle's / Aunt's son or daughter | Cousin |
| Son's wife | Daughter-in-law |
| Daughter's husband | Son-in-law |
| Brother's wife | Sister-in-law |
| Sister's husband | Brother-in-law |
| Spouse's mother | Mother-in-law |
| Spouse's father | Father-in-law |
| Spouse's brother | Brother-in-law |
| Sibling's son | Nephew |
| Sibling's daughter | Niece |

## 🔑 Master Summary

| Rule | When to use |
|------|------------|
| Draw family tree | Every blood relation question |
| Work backwards from the complex phrase | "Pointing to a photo" type |
| Replace symbols first | Coded relation type |
| Note gender carefully | Son/daughter/brother/sister clues |
| "Only son/daughter" = self | Common exam trick |
