## 📐 The Most Frequently Tested Formulas (Know These Cold)

| Formula | Topic |
|---------|-------|
| `(XY)/(X+Y)` | Two people/pipes working together |
| `Distance = Speed × Time` | TSD — always first |
| `Downstream = B + S; Upstream = B − S` | Boats |
| `Sum of train+platform` | Train crossing |
| `SP = CP × (100+P%)/100` | Profit/Loss |
| `CP = SP × 100/(100+P%)` | Finding CP from SP |
| `SI = PRT/100` | All 4 derived forms |
| `A = P(1+R/100)^T` | Compound Interest |
| Half-yearly: R→R/2, T→2T | CI compounding |
| `Net% = a+b+ab/100` | Successive % change |
| `(n+1)/2` | Average of 1 to n |
| New member = `New avg × New n − Old avg × Old n` | Average change |
| `LCM × HCF = product of two numbers` | HCF-LCM relation |
| `nCr = n!/(r!(n-r)!)` | Combination |
| `nPr = n!/(n-r)!` | Permutation |
| `(n-1)!` | Circular arrangement |
| `n(A∪B) = n(A)+n(B)−n(A∩B)` | Venn diagrams |
| `|A| = ad−bc` | 2×2 Determinant |
| `Σ(fx)/Σf` | Frequency mean |
| Median = `L+[(n/2−cf)/f]×h` | Grouped data |
