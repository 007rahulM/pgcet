## SECTION A: Mathematical Aptitude

**Q1.** A and B together can complete a work in 8 days. A alone takes 12 days. How many days does B alone take?
- (A) 20 days  (B) 24 days  (C) 28 days  (D) 30 days

**Answer: (B) 24 days**
**Solution:**
- A+B together in 8 days → combined efficiency = 1/8
- A alone in 12 days → A's efficiency = 1/12
- B's efficiency = 1/8 − 1/12 = 3/24 − 2/24 = 1/24
- B alone = **24 days**

---

**Q2.** A car travels 120 km at 60 km/h and next 120 km at 40 km/h. Average speed?
- (A) 50 km/h  (B) 48 km/h  (C) 45 km/h  (D) 52 km/h

**Answer: (B) 48 km/h**
**Solution:**
- Equal distances → Average speed = 2ab/(a+b) = 2×60×40/(60+40) = 4800/100 = **48 km/h**

**Q3.** Principal ₹10000 at 10% p.a. CI for 2 years. Find amount.
- (A) ₹11000  (B) ₹12000  (C) ₹12100  (D) ₹11500

**Answer: (C) ₹12100**
**Solution:**
- A = P(1 + R/100)^T = 10000 × (1.1)² = 10000 × 1.21 = **₹12100**

**Q4.** In a bag: 3 red, 4 blue, 5 green balls. Probability of picking a blue ball?
- (A) 1/3  (B) 1/4  (C) 4/12  (D) 5/12

**Answer: (C) 4/12 = 1/3**
**Solution:**
- Total = 3+4+5 = 12 balls
- P(blue) = 4/12 = **1/3**

**Q5.** Convert hexadecimal AF₁₆ to decimal.
- (A) 165  (B) 175  (C) 185  (D) 195

**Answer: (B) 175**
**Solution:**
- A = 10, F = 15
- 10×16 + 15×1 = 160 + 15 = **175**

**Q6.** Which sorting algorithm has best average case O(n log n)?
- (A) Bubble Sort  (B) Quick Sort  (C) Selection Sort  (D) Insertion Sort

**Answer: (B) Quick Sort**
**Solution:** Quick Sort average = O(n log n). Bubble, Selection, Insertion = O(n²) average.

**Q7.** p = T, q = F. Evaluate p → q.
- (A) True  (B) False  (C) Cannot determine  (D) Both

**Answer: (B) False**
**Solution:** Implication (→) is FALSE only when p=True and q=False. Here p=T, q=F → **FALSE**

**Q8.** Find determinant of ┌5 2┐
                             └3 4┘
- (A) 14  (B) 20  (C) 26  (D) 26

**Answer: (A) 14**
**Solution:** det = 5×4 − 2×3 = 20 − 6 = **14**

**Q9.** Find mean of first 10 natural numbers.
- (A) 5  (B) 5.5  (C) 6  (D) 4.5

**Answer: (B) 5.5**
**Solution:**
- Sum = 1+2+...+10 = 55
- Mean = 55/10 = **5.5**

**Q10.** In set A = {1,2,3,4,5}, B = {3,4,5,6,7}. Find A−B.
- (A) {1,2}  (B) {6,7}  (C) {3,4,5}  (D) {1,2,6,7}

**Answer: (A) {1,2}**
**Solution:** A−B = elements in A but NOT in B = {1,2}

## SECTION B: Computer Awareness

**Q11.** Which is NOT a characteristic of RAM?
- (A) Volatile  (B) Random access  (C) Stores OS permanently  (D) Read/write

**Answer: (C) Stores OS permanently**
**Solution:** RAM is volatile — it doesn't store data permanently. OS is loaded INTO RAM from ROM/disk when computer boots. ROM stores the bootloader permanently.

---

**Q12.** Inorder traversal of a BST gives:
- (A) Pre-sorted order  (B) Sorted descending  (C) Sorted ascending  (D) Random

**Answer: (C) Sorted ascending**
**Solution:** Inorder (Left→Root→Right) on a BST always produces sorted ascending output.

---

**Q13.** Which OSI layer adds IP address?
- (A) Data Link  (B) Network  (C) Transport  (D) Physical

**Answer: (B) Network**
**Solution:** Layer 3 (Network) handles IP addressing and routing. Layer 2 handles MAC addresses.

---

**Q14.** In C, `++i` vs `i++` — which increments BEFORE using the value?
- (A) i++  (B) ++i  (C) Both same  (D) Neither

**Answer: (B) ++i**
**Solution:** `++i` (prefix) = increment THEN use. `i++` (postfix) = use THEN increment.

---

**Q15.** Primary key must be:
- (A) Numeric only  (B) Unique and Not Null  (C) Indexed  (D) Foreign reference

**Answer: (B) Unique and Not Null**
**Solution:** Primary key = uniquely identifies each row + cannot be NULL.

**Q16.** SMTP protocol is used for:
- (A) File transfer  (B) Web browsing  (C) Email sending  (D) Remote login

**Answer: (C) Email sending**
**Solution:** SMTP = Simple Mail Transfer Protocol (port 25). Used for sending emails.

**Q17.** What is 2's complement of 1011₂?
- (A) 0100  (B) 0101  (C) 0110  (D) 1010

**Answer: (B) 0101**
**Solution:**
- 1011
- 1's complement: 0100
- Add 1: 0100 + 1 = **0101**

**Q18.** Which scheduling algorithm can cause starvation?
- (A) Round Robin  (B) FCFS  (C) Priority  (D) Shortest Job First

**Answer: (C) Priority**
**Solution:** Priority scheduling can cause "starvation" — low priority processes may never get CPU if high-priority processes keep arriving. Round Robin and FCFS give everyone a turn.

---

**Q19.** Which of these is an output of a Compiler?
- (A) Source code  (B) Object code  (C) Assembly code  (D) Machine code only

**Answer: (B) Object code**
**Solution:** Compiler translates source code → object code (binary). Linker then combines object files → executable.

**Q20.** What is the port number of HTTP?
- (A) 21  (B) 25  (C) 80  (D) 443

**Answer: (C) 80**
**Solution:** HTTP = port 80. HTTPS = 443. FTP = 21. SMTP = 25.

## SECTION C: Analytical & Logical Reasoning

**Q21.** If "TIGER" is coded as "QDFBP" (each letter shifted back by 3), how is "LION" coded?
- (A) IFKL  (B) IFLL  (C) JGLK  (D) IELK

**Answer: (A) IFKL**
**Solution:**
- L-3=I, I-3=F, O-3=L, N-3=K → **IFLK**
Wait — recheck: L(12)−3=9=I, I(9)−3=6=F, O(15)−3=12=L, N(14)−3=11=K → **IFLK**

**Answer: IFLK** (Note: verify against actual paper — pattern is subtract 3)

**Q22.** A is B's sister. C is B's mother. D is C's father. E is D's mother. How is A related to D?
- (A) Granddaughter  (B) Great-granddaughter  (C) Daughter  (D) Grandmother

**Answer: (A) Granddaughter**
**Solution:**
- C is B's mother → C is also A's mother
- D is C's father → D is A's maternal grandfather
- A is D's granddaughter ✅

**Q23.** Series: 3, 9, 27, 81, ?
- (A) 162  (B) 243  (C) 324  (D) 180

**Answer: (B) 243**
**Solution:** Each term ×3: 3×3=9, 9×3=27, 27×3=81, 81×3=**243**

---

**Q24.** 6 people sit in a circular arrangement. How many ways can they sit?
- (A) 720  (B) 120  (C) 24  (D) 360

**Answer: (B) 120**
**Solution:** Circular arrangements of n people = (n−1)! = (6−1)! = 5! = **120**

---

**Q25.** All roses are flowers. Some flowers fade. Conclusion: Some roses fade.
- (A) Definitely true  (B) Definitely false  (C) Possibly true  (D) Data insufficient

**Answer: (C) Possibly true**
**Solution:** "Some flowers fade" doesn't specify which flowers. Roses might or might not be among them. The conclusion POSSIBLY follows but is not definite.
