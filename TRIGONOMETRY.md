### 1. STANDARD TRIGONOMETRIC VALUES

**The MOST IMPORTANT Table - MEMORIZE THIS!**

| Angle | 0° | 30° | 45° | 60° | 90° |
|-------|-----|-----|-----|-----|-----|
| **Radians** | 0 | π/6 | π/4 | π/3 | π/2 |
| **sin** | 0 | 1/2 | 1/√2 | √3/2 | 1 |
| **cos** | 1 | √3/2 | 1/√2 | 1/2 | 0 |
| **tan** | 0 | 1/√3 | 1 | √3 | ∞ |
| **cot** | ∞ | √3 | 1 | 1/√3 | 0 |
| **sec** | 1 | 2/√3 | √2 | 2 | ∞ |
| **csc** | ∞ | 2 | √2 | 2/√3 | 1 |

**60-Second Memory Trick (sin values):**
```
sin 0° = √0/2 = 0
sin 30° = √1/2 = 1/2
sin 45° = √2/2 = 1/√2
sin 60° = √3/2
sin 90° = √4/2 = 1

Pattern: √0, √1, √2, √3, √4 all divided by 2!
```

**For cos:** Just REVERSE the sin values!
```
cos 0° = 1
cos 30° = √3/2
cos 45° = 1/√2
cos 60° = 1/2
cos 90° = 0
```

### 2. FUNDAMENTAL IDENTITIES

**Pythagorean Identities:**
```
sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = csc²θ
```

**Reciprocal Identities:**
```
csc θ = 1/sin θ
sec θ = 1/cos θ
cot θ = 1/tan θ
```

**Quotient Identities:**
```
tan θ = sin θ/cos θ
cot θ = cos θ/sin θ
```

---

### 3. COMPOUND ANGLE FORMULAS

**Sum and Difference:**
```
sin(A + B) = sin A cos B + cos A sin B
sin(A - B) = sin A cos B - cos A sin B

cos(A + B) = cos A cos B - sin A sin B
cos(A - B) = cos A cos B + sin A sin B

tan(A + B) = (tan A + tan B)/(1 - tan A tan B)
tan(A - B) = (tan A - tan B)/(1 + tan A tan B)
```

**60-Second Trick:**
- Sin: Same (SC + CS)
- Cos: Change sign (CC - SS)

---

### 4. DOUBLE ANGLE FORMULAS

```
sin 2A = 2 sin A cos A
cos 2A = cos²A - sin²A = 2cos²A - 1 = 1 - 2sin²A
tan 2A = 2tan A/(1 - tan²A)
```

### 5. INVERSE TRIGONOMETRIC FUNCTIONS

**Principal Value Ranges (CRITICAL!):**

| Function | Range |
|----------|-------|
| sin⁻¹(x) | [-π/2, π/2] |
| cos⁻¹(x) | [0, π] |
| tan⁻¹(x) | (-π/2, π/2) |
| cot⁻¹(x) | (0, π) |
| sec⁻¹(x) | [0, π] - {π/2} |
| csc⁻¹(x) | [-π/2, π/2] - {0} |

**Important Values:**

| x | sin⁻¹(x) | cos⁻¹(x) | tan⁻¹(x) |
|---|----------|----------|----------|
| 0 | 0 | π/2 | 0 |
| 1/2 | π/6 | π/3 | π/6 |
| 1/√2 | π/4 | π/4 | π/4 |
| √3/2 | π/3 | π/6 | π/3 |
| 1 | π/2 | 0 | π/4 |

**60-Second Trick:**
```
sin⁻¹(x) + cos⁻¹(x) = π/2
tan⁻¹(x) + cot⁻¹(x) = π/2
```

---

### 6. SPECIAL ANGLE CONVERSIONS

**Negative Angles:**
```
sin(-θ) = -sin θ
cos(-θ) = cos θ
tan(-θ) = -tan θ
```

**Complementary Angles (π/2 - θ):**
```
sin(π/2 - θ) = cos θ
cos(π/2 - θ) = sin θ
tan(π/2 - θ) = cot θ
```

**Supplementary Angles (π - θ):**
```
sin(π - θ) = sin θ
cos(π - θ) = -cos θ
tan(π - θ) = -tan θ
```

## 🔥 PRACTICE PROBLEMS (EXAM STYLE)

### PROBLEM 1: Standard Value

**Question:** Find the value of sin(π/6).

**Solution:**
```
π/6 radians = 30°
From table: sin 30° = 1/2
```

**Answer: 1/2**

---

### PROBLEM 2: Inverse Trig

**Question:** Find cos⁻¹(1/2).

**Solution:**
```
We need angle θ such that cos θ = 1/2
From table: cos 60° = cos(π/3) = 1/2
Since π/3 is in range [0, π], answer is π/3
```

**Answer: π/3**

---

### PROBLEM 3: sin⁻¹(sin θ) Type

**Question:** Find sin⁻¹(sin 3π/5).

**Solution:**
```
3π/5 is NOT in range [-π/2, π/2]
3π/5 = π - 2π/5 (supplementary angle)
sin(3π/5) = sin(π - 2π/5) = sin(2π/5)
2π/5 IS in range [-π/2, π/2]
```

**Answer: 2π/5**

---

### PROBLEM 4: Expression Simplification

**Question:** Find value of 3sin(π/6)sec(π/3) - 4sin(5π/6)cot(π/4).

**Solution:**
```
Step 1: Find each value
sin(π/6) = 1/2
sec(π/3) = 1/cos(π/3) = 1/(1/2) = 2
sin(5π/6) = sin(π - π/6) = sin(π/6) = 1/2
cot(π/4) = 1/tan(π/4) = 1/1 = 1

Step 2: Calculate
3 × (1/2) × 2 = 3
4 × (1/2) × 1 = 2

Step 3: Final answer
3 - 2 = 1
```

**Answer: 1**

---

### PROBLEM 5: Compound Angle

**Question:** Find sin(π/4 + π/6).

**Solution:**
```
Using: sin(A + B) = sin A cos B + cos A sin B

sin(π/4 + π/6) = sin(π/4)cos(π/6) + cos(π/4)sin(π/6)
                = (1/√2)(√3/2) + (1/√2)(1/2)
                = √3/(2√2) + 1/(2√2)
                = (√3 + 1)/(2√2)
```

**Answer: (√3 + 1)/(2√2)**

---

### PROBLEM 6: tan⁻¹(1)

**Question:** Find tan⁻¹(1).

**Solution:**
```
We need angle θ such that tan θ = 1
From table: tan 45° = tan(π/4) = 1
Since π/4 is in range (-π/2, π/2), answer is π/4
```

**Answer: π/4**

---

### PROBLEM 7: Identity Application

**Question:** If sin θ = 3/5, find cos θ.

**Solution:**
```
Using: sin²θ + cos²θ = 1
(3/5)² + cos²θ = 1
9/25 + cos²θ = 1
cos²θ = 1 - 9/25 = 16/25
cos θ = ±4/5

If θ in first quadrant, cos θ = 4/5
```

**Answer: 4/5 (assuming first quadrant)**

---

### PROBLEM 8: Degree to Radian

**Question:** Convert 120° to radians.

**Solution:**
```
Radians = Degrees × (π/180)
        = 120 × (π/180)
        = 2π/3
```

**Answer: 2π/3**

---

### PROBLEM 9: cot⁻¹(√3)

**Question:** Find cot⁻¹(√3).

**Solution:**
```
We need angle θ such that cot θ = √3
From table: cot 30° = cot(π/6) = √3
Since π/6 is in range (0, π), answer is π/6
```

**Answer: π/6**

---

### PROBLEM 10: Double Angle

**Question:** If sin θ = 1/2, find sin 2θ.

**Solution:**
```
Using: sin 2θ = 2 sin θ cos θ

First find cos θ:
sin²θ + cos²θ = 1
(1/2)² + cos²θ = 1
cos²θ = 3/4
cos θ = √3/2

Now: sin 2θ = 2 × (1/2) × (√3/2) = √3/2
```

**Answer: √3/2**

## 🎓 MEMORY TRICKS

### Trick 1: Sin Values (0° to 90°)
```
0° → √0/2 = 0
30° → √1/2 = 1/2
45° → √2/2 = 1/√2
60° → √3/2
90° → √4/2 = 1
```
**Just count: √0, √1, √2, √3, √4 and divide by 2!**

### Trick 2: Cos Values
**Reverse the sin values!**
```
cos 0° = sin 90° = 1
cos 30° = sin 60° = √3/2
cos 45° = sin 45° = 1/√2
cos 60° = sin 30° = 1/2
cos 90° = sin 0° = 0
```

### Trick 3: Inverse Trig Ranges
```
sin⁻¹ and tan⁻¹: Center at 0, range includes negative
cos⁻¹ and cot⁻¹: Start from 0, all positive
```

### Trick 4: Principal Values
**"If asked sin⁻¹(sin θ), first check if θ is in [-π/2, π/2]"**
- If YES → answer is θ
- If NO → find equivalent angle in range
