## Key Formulas

| Formula | Use |
|---------|-----|
| n(A ∪ B) = n(A) + n(B) − n(A ∩ B) | Two sets union |
| n(A ∩ B) = n(A) + n(B) − n(A ∪ B) | Find intersection |
| Only A = n(A) − n(A ∩ B) | Elements in A only |
| Neither = Total − n(A ∪ B) | Neither set |
| 2ⁿ subsets | Power set |
| \|A × B\| = \|A\| × \|B\| | Cartesian product size |

## Types of Relations (Quick Reference)

- **Reflexive:** Every element relates to itself — (a,a) ∈ R
- **Symmetric:** (a,b) ∈ R → (b,a) ∈ R
- **Transitive:** (a,b) and (b,c) ∈ R → (a,c) ∈ R
- **Equivalence:** All three above
