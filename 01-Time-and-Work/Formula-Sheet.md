## Core Formulas

- Work = Rate × Time
- If A finishes in x days, A's 1-day work = 1/x
- Together time of A and B = \(\frac{xy}{x+y}\) (when both start together)
- For efficiency: If efficiency ratio A:B = m:n, then time ratio A:B = n:m
