## âšˇ 60-Second Shortcut

For analogy coding:
1. Write letters **one below another**
2. Record shift at each position
3. Check in this order:
   - constant shift
   - reverse
   - opposite letters
   - odd/even split
   - +1, +2, +3... progression
4. Only then apply to the new word

**Golden rule:** Tough coding questions are usually solved by a **table**, not by guesswork.
