# Regression & Prediction — Pattern 6

## 🔍 When This Comes Up
- "Predict y from x" or "regression line of y on x"
- Given mean and regression coefficient
- Questions involving best-fit line interpretation

---

## 🧠 Core Idea
Use regression lines to **estimate one variable** from the other using slope and mean values.

---

## 📐 Key Formulas
Regression of y on x:
```
y - ȳ = b_yx (x - x̄)
```
Regression of x on y:
```
x - x̄ = b_xy (y - ȳ)
```
Where:
```
b_yx = r (σ_y / σ_x)
```

---

## ✅ Worked Example

**❓ Question:** If x̄ = 10, ȳ = 20, and b_yx = 1.5, estimate y when x = 14.

**✏️ My Solution:**
```
y - 20 = 1.5(14 - 10) = 1.5 × 4 = 6
y = 26
```
**✅ Answer:** 26

---

## 📝 Practice
1. If x̄ = 5, ȳ = 12, b_yx = 2, find y when x = 8
2. If r = 0.5, σ_x = 4, σ_y = 6, find b_yx

**Answers:**
1) y = 18
2) b_yx = 0.5 × (6/4) = 0.75
