## Quick Reference

| Measure | Formula |
|---------|---------|
| Mean | Σx / n |
| Median (odd) | Value at position (n+1)/2 after sorting |
| Median (even) | Average of n/2 and (n/2+1) positions |
| Mode | Most frequent value |
| Range | Max − Min |
| Variance | Σ(x − x̄)² / n |
| SD | √Variance |

## Key Properties

- Adding constant c: Mean/Median/Mode all increase by c, SD unchanged
- Multiplying by k: Mean ×k, SD ×|k|, Variance ×k²
- All values equal: SD = 0
