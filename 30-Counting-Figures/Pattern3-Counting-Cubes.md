# Counting Cubes — Pattern 3

## 🧠 Core Idea

Count both **unit cubes** and **all larger cubes** in a block.

## ✅ Key Formulas

**Unit cubes in an n×n×n cube:**
```
Total unit cubes = n³
```

**All cubes of every size in an n×n×n cube:**
```
1³ + 2³ + 3³ + ... + n³ = [n(n+1)/2]²
```

## 📌 Example (3×3×3 Cube)

- Unit cubes = 3³ = 27
- All cubes of all sizes = 1³ + 2³ + 3³ = 36

## 🔥 Example Question

**❓ Question:** How many cubes of all sizes are in a 3×3×3 cube?

**💡 What I'll use:** 1³ + 2³ + 3³

**✏️ My Solution:**

Step 1: 1³ = 1

Step 2: 2³ = 8

Step 3: 3³ = 27

Step 4: Total = 1 + 8 + 27 = 36

**✅ Answer: 36**
