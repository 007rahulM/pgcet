# Simplification Strategies — Answers

### Q1
```
A(A + B) = A
```
**Answer:** A

---

### Q2
```
A + A' = 1
```
**Answer:** 1

---

### Q3
```
(A·B)' = A' + B'
```
**Answer:** A' + B'

---

### Q4
```
A·B + A·B' = A(B + B') = A
```
**Answer:** A

---

### Q5
```
(A + B)(A + B') = A + BB' = A
```
**Answer:** A
