# Profit & Loss — Formula Sheet

| If question says... | Use this formula/rule | Go to examples |
|---|---|---|
| Find profit/loss amount | Profit = SP−CP, Loss = CP−SP | [01-Basic-Concepts.md](./01-Basic-Concepts.md) |
| Find profit/loss percent | Profit% = (Profit/CP)×100, Loss% = (Loss/CP)×100 | [01-Basic-Concepts.md](./01-Basic-Concepts.md) |
| Marked price and discount | Discount = MP−SP, Discount% = (Discount/MP)×100 | [02-Discount.md](./02-Discount.md) |
| Two discounts in sequence | Net discount% = a+b−ab/100 | [03-Successive-Discounts.md](./03-Successive-Discounts.md) |
| Dishonest dealer/false weight | Compare paid price vs actual quantity delivered | [04-Dishonest-Dealer.md](./04-Dishonest-Dealer.md) |

## Core Formulas

- SP = CP × (100+Profit%)/100
- SP = CP × (100−Loss%)/100
