## Quick Decision Table

| Premise Combo | Conclusion |
|--------------|-----------|
| All + All | All (chained) |
| All + No | No |
| Some + All | Some |
| Some + No | Some Not |
| Some + Some | Nothing definite |
| All + Some | Nothing definite |

## Valid Reversals

- "All A are B" → "Some B are A" ✅
- "No A is B" → "No B is A" ✅
- "Some A are B" → "Some B are A" ✅
- "Some A are not B" → No valid reversal ✗
