# Two-Statement Syllogisms — Practice Problem Solutions

### Set 1

**❓ Question:** Statements: All birds fly. All flying things have wings.
Conclusions: I. All birds have wings. II. All things with wings are birds.

**🤔 What I understood:**
- Given: All birds fly; All flying things have wings
- Find: Which conclusions follow?

**💡 What I'll use:** Syllogism rule — All + All = All (same chain direction)

**✏️ My Solution:**

Step 1: Check Conclusion I
- All birds fly + All flying things have wings
- Middle term = "flying things/fly"
- All + All = All → All birds have wings ✅ → Conclusion I follows

Step 2: Check Conclusion II
- "All things with wings are birds" — this reverses the chain
- Valid reversal of "All A are B" gives only "Some B are A", not "All B are A"
- We can only say "Some winged things are birds", not "All" ✗ → Conclusion II does not follow

**✅ Answer: Only I follows**

### Set 4

**❓ Question:** Statements: All mangoes are fruits. No fruit is a vegetable.
Conclusions: I. No mango is a vegetable. II. Some vegetables are mangoes.

**🤔 What I understood:**
- Given: All mangoes are fruits; No fruit is a vegetable
- Find: Which conclusions follow?

**💡 What I'll use:** Syllogism rule — All + No = No

**✏️ My Solution:**

Step 1: Check Conclusion I
- All mangoes are fruits + No fruit is a vegetable
- Middle term = "fruits"
- All + No = No → No mango is a vegetable ✅ → Conclusion I follows

Step 2: Check Conclusion II
- "Some vegetables are mangoes" contradicts "No mango is a vegetable"
- Since we concluded no mango is a vegetable, no vegetable can be a mango ✗ → Conclusion II does not follow

**✅ Answer: Only I follows**

### Set 5

**❓ Question:** Statements: Some trains are buses. Some buses are taxis.
Conclusions: I. Some trains are taxis. II. Some taxis are trains.

**🤔 What I understood:**
- Given: Some trains are buses; Some buses are taxis
- Find: Which conclusions follow?

**💡 What I'll use:** Syllogism rule — Some + Some = No definite conclusion

**✏️ My Solution:**

Step 1: Apply the rule
- Some trains are buses + Some buses are taxis
- Rule: Some + Some = No definite conclusion
- The overlap between trains and buses may have no intersection with the overlap between buses and taxis

Step 2: Check both conclusions
- Conclusion I: Some trains are taxis — cannot be concluded ✗
- Conclusion II: Some taxis are trains — also cannot be concluded ✗ (same reasoning, just reversed)

**✅ Answer: Neither I nor II follows**
