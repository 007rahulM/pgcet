# Either-Or and Three-Statement Syllogisms — Practice Problem Solutions

### Set 1

**❓ Question:** Statements: Some books are pens. No pen is a cap.
Conclusions: I. No book is a cap. II. Some books are not caps.

**🤔 What I understood:**
- Given: Some books are pens; No pen is a cap
- Find: Which conclusions follow?

**💡 What I'll use:** Syllogism rule — Some + No = Some Not

**✏️ My Solution:**

Step 1: Apply the rule
- Some books are pens + No pen is a cap
- Middle term = "pens"
- Some + No = Some Not → Some books are not caps ✅ → Conclusion II follows

Step 2: Check Conclusion I
- "No book is a cap" is a universal negative — it claims ALL books are not caps
- We only established that SOME books are not caps (those that are pens)
- Other books (not pens) could still be caps ✗ → Conclusion I is too strong and does not follow

**✅ Answer: Only II follows**

### Set 2

**❓ Question:** Statements: All birds are animals. All animals are living. Some living things are moving.
Conclusions: I. All birds are living. II. Some moving things are animals. III. All animals are moving.

**🤔 What I understood:**
- Given: Three statements chained together
- Find: Which of the three conclusions follow?

**💡 What I'll use:** Chain the first two statements, then test each conclusion

**✏️ My Solution:**

Step 1: Check Conclusion I
- All birds are animals + All animals are living
- All + All = All → All birds are living ✅ → Conclusion I follows

Step 2: Check Conclusion III
- The third statement says: Some living things are moving
- We know all animals are living, but does that mean all animals are moving?
- Some living are moving — but animals are a subset of living; we cannot conclude ALL animals move ✗ → Conclusion III does not follow

Step 3: Check Conclusion II
- "Some moving things are animals" would require knowing that the moving things overlap with animals
- Some living are moving; animals are living, but the "some moving" subset might not include any animals ✗ → Conclusion II cannot be definitively concluded

**✅ Answer: Only I follows**

### Set 3

**❓ Question:** Statements: Some chairs are tables. Some tables are wooden.
Conclusions: I. Some chairs are wooden. II. No chair is wooden.

**🤔 What I understood:**
- Given: Some chairs are tables; Some tables are wooden
- Find: Which conclusions follow? (Either-Or scenario)

**💡 What I'll use:** Some + Some = no definite conclusion; then check for complementary pair

**✏️ My Solution:**

Step 1: Apply the rule
- Some chairs are tables + Some tables are wooden
- Rule: Some + Some = No definite conclusion
- Neither I nor II follows individually from the syllogism

Step 2: Check for complementary pair (Either-Or rule)
- Conclusion I says: "Some chairs are wooden" (possibility)
- Conclusion II says: "No chair is wooden" (complete negation of the same thing)
- These are exact opposites about the same subject and object
- In reality, one of them MUST be true — chairs either have some wooden members or they don't

Step 3: Apply Either-Or rule
- When neither conclusion follows but they form a complementary pair → "Either I or II follows" ✅

**✅ Answer: Either I or II follows**

### Set 4

**❓ Question:** Statements: All engineers are graduates. Some graduates are rich. No rich person is poor.
Conclusions: I. Some engineers are rich. II. Some graduates are not poor. III. Some engineers are not poor.

**🤔 What I understood:**
- Given: Three statements about engineers, graduates, rich, and poor people
- Find: Which conclusions follow?

**💡 What I'll use:** Apply syllogism rules to pairs of statements; check direction of middle terms

**✏️ My Solution:**

Step 1: Check Conclusion I
- All engineers are graduates + Some graduates are rich
- Middle term = "graduates", but direction matters: All A→B, Some B→C
- All + Some (in reverse direction) = **No definite conclusion** ✗ → Conclusion I does not follow

Step 2: Check Conclusion II
- Some graduates are rich + No rich person is poor
- Middle term = "rich"
- Some + No = Some Not → Some graduates are not poor ✅ → Conclusion II follows

Step 3: Check Conclusion III
- For "Some engineers are not poor," we would need to connect engineers to "not poor"
- Since Conclusion I failed (we can't say some engineers are rich), we cannot chain to "not poor" either ✗ → Conclusion III does not follow

**✅ Answer: Only II follows**
