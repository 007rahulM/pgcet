# Pattern 1: Word Analogy

## 🔍 How to Recognize This Pattern

- "Book : Library :: Painting : ___"
- "Doctor : Hospital :: Teacher : ___"
- "A is to B as C is to ___"
- The `:` means "is related to" and `::` means "in the same way as"

---

## 🧠 The Golden Rule

> **Step 1: Identify the relationship between the first pair.**
> **Step 2: Apply the EXACT SAME relationship to the second pair.**

---

## 📐 Key Relationship Types

### Type 1: Object : Place (Where it is kept / used)
| First Pair | Second Pair |
|-----------|-------------|
| Book : Library | Painting : Gallery |
| Patient : Hospital | Prisoner : Jail |
| Money : Bank | Crops : Farm |
| Fish : Water | Bird : Sky |

### Type 2: Part : Whole (The item is a part of)
| First Pair | Second Pair |
|-----------|-------------|
| Page : Book | Chapter : Novel |
| Petal : Flower | Branch : Tree |
| Wheel : Car | Key : Keyboard |
| Finger : Hand | Toe : Foot |

### Type 3: Worker : Tool (Person uses this tool)
| First Pair | Second Pair |
|-----------|-------------|
| Carpenter : Saw | Tailor : Needle |
| Painter : Brush | Doctor : Stethoscope |
| Farmer : Plough | Chef : Knife |
| Architect : Blueprint | Musician : Instrument |

### Type 4: Thing : Category (Item belongs to this group)
| First Pair | Second Pair |
|-----------|-------------|
| Mango : Fruit | Potato : Vegetable |
| Python : Language | HTML : Markup |
| Stack : Data Structure | SQL : Query Language |
| January : Month | Monday : Day |

### Type 5: Antonym (Opposites)
| First Pair | Second Pair |
|-----------|-------------|
| Hot : Cold | Day : Night |
| Tall : Short | Wide : Narrow |
| Birth : Death | Joy : Sorrow |
| Open : Close | Buy : Sell |

### Type 6: Degree/Intensity
| First Pair | Second Pair |
|-----------|-------------|
| Warm : Hot | Cool : Cold |
| Dislike : Hate | Like : Love |
| Tired : Exhausted | Annoyed : Furious |

### Type 7: Function (What it does)
| First Pair | Second Pair |
|-----------|-------------|
| Pen : Write | Knife : Cut |
| Camera : Click | Watch : Time |
| Accelerator : Speed | Brake : Stop |

### Type 8: Product : Source (Made from)
| First Pair | Second Pair |
|-----------|-------------|
| Butter : Milk | Curd : Milk |
| Wood : Tree | Leather : Animal |
| Gold : Mine | Pearl : Oyster |

## ✅ Step-by-Step Examples

### Example 1

**❓ Question:** Doctor : Hospital :: Teacher : ?
- (A) Office  (B) School  (C) College  (D) Library

**🤔 What I understood:**
- Doctor works at Hospital (worker : workplace)
- Find: Where does Teacher work?

**✏️ My Solution:**
- Relationship: Worker → Workplace
- Doctor works at Hospital ✓
- Teacher works at School ✓

**✅ Answer: (B) School**

---

### Example 2

**❓ Question:** Book : Library :: Painting : ?
- (A) Museum  (B) Gallery  (C) Auction  (D) Studio

**🤔 What I understood:**
- Book is stored/found in Library (object : place it is kept)
- Find: Where is Painting kept/displayed?

**✏️ My Solution:**
- Relationship: Object → Place where it is displayed/kept
- Books are in Library, Paintings are in Gallery ✓

**✅ Answer: (B) Gallery**

---

### Example 3

**❓ Question:** Petal : Flower :: Page : ?
- (A) Notebook  (B) Chapter  (C) Book  (D) Library

**🤔 What I understood:**
- Petal is a PART of Flower (part : whole)
- Find: What is Page a part of?

**✏️ My Solution:**
- Relationship: Part → Whole
- Petal is part of Flower ✓
- Page is part of Book ✓

**✅ Answer: (C) Book**

---

### Example 4

**❓ Question:** Python : Programming Language :: Stack : ?
- (A) Algorithm  (B) Data Structure  (C) Database  (D) Network

**🤔 What I understood:**
- Python is a type of Programming Language (thing : category)
- Find: What category does Stack belong to?

**✏️ My Solution:**
- Relationship: Specific item → Category it belongs to
- Python is a Programming Language ✓
- Stack is a Data Structure ✓

**✅ Answer: (B) Data Structure**

---

### Example 5

**❓ Question:** Carpenter : Saw :: Farmer : ?
- (A) Field  (B) Plough  (C) Crop  (D) Tractor

**🤔 What I understood:**
- Carpenter USES a Saw (worker : tool)
- Find: What does Farmer use as a tool?

**✏️ My Solution:**
- Relationship: Worker → Tool used
- Carpenter uses Saw ✓
- Farmer uses Plough ✓

**✅ Answer: (B) Plough**

---

### Example 6

**❓ Question:** Hot : Cold :: Tall : ?
- (A) Big  (B) Long  (C) Short  (D) Slim

**🤔 What I understood:**
- Hot and Cold are OPPOSITES (antonyms)
- Find: Opposite of Tall

**✏️ My Solution:**
- Relationship: Antonym pair
- Hot ↔ Cold ✓
- Tall ↔ Short ✓

**✅ Answer: (C) Short**

---

## ⚡ 60-Second Method

1. **Read the first pair carefully** — find their relationship in 5 seconds
2. **Name the relationship** — "worker : tool", "part : whole", etc.
3. **Apply the same relationship** to the second pair
4. **Check all options** — if two seem right, think more precisely about the relationship

## 📝 Practice Problems

**Q1.** Fish : Water :: Bird : ?
- (A) Tree  (B) Nest  (C) Sky  (D) Feather

**Q2.** Keyboard : Computer :: Steering Wheel : ?
- (A) Driver  (B) Car  (C) Road  (D) Engine

**Q3.** Mango : Fruit :: SQL : ?
- (A) Table  (B) Database  (C) Query Language  (D) Programming Language

**Q4.** Pearl : Oyster :: Leather : ?
- (A) Shoe  (B) Animal  (C) Skin  (D) Factory

**Q5.** Dislike : Hate :: Tired : ?
- (A) Sleep  (B) Rest  (C) Exhausted  (D) Weak

**Q6.** Nurse : Hospital :: Pilot : ?
- (A) Airplane  (B) Airport  (C) Cockpit  (D) Sky

---

## ✔️ Answers

| Q | Answer | Reason |
|---|--------|--------|
| Q1 | (C) Sky | Fish lives in Water :: Bird lives in Sky — habitat |
| Q2 | (B) Car | Keyboard controls Computer :: Steering Wheel controls Car — control device |
| Q3 | (C) Query Language | Mango is a Fruit :: SQL is a Query Language — thing : category |
| Q4 | (B) Animal | Pearl comes from Oyster :: Leather comes from Animal — product : source |
| Q5 | (C) Exhausted | Dislike→Hate (increase in degree) :: Tired→Exhausted (increase in degree) |
| Q6 | (B) Airport | Nurse works at Hospital :: Pilot works at Airport — worker : workplace |
