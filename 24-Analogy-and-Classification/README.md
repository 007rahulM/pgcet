# Analogy & Classification — Overview

Analogy and Classification questions appear in **2–3 questions** in MCA PGCET every year. They test your ability to find relationships between pairs of words/numbers/concepts.

## Types of Analogy Relationships

1. **Object : Place** (Book → Library, Patient → Hospital)
2. **Part : Whole** (Page → Book, Petal → Flower)
3. **Worker : Tool** (Carpenter → Saw, Painter → Brush)
4. **Thing : Category** (Mango → Fruit, Python → Language)
5. **Antonym Pair** (Hot → Cold, Day → Night)
6. **Synonym Pair** (Happy → Joyful)
7. **Degree** (Warm → Hot, Dislike → Hate)
8. **Function** (Pen → Write, Knife → Cut)
9. **Number Pattern** (4 → 16 means square, 3 → 9 means square)

## The Golden Rule

> **Find the RELATIONSHIP in the first pair, then apply the SAME relationship to the second pair.**
