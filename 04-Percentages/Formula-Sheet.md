# Percentages — Formula Sheet

| If question says... | Use this formula/rule | Go to examples |
|---|---|---|
| What % of whole is part? | Percentage = (Part/Whole)×100 | [01-Basic-Calculations.md](./01-Basic-Calculations.md) |
| Find part from % and whole | Part = (Percentage×Whole)/100 | [01-Basic-Calculations.md](./01-Basic-Calculations.md) |
| Increase/decrease by x% | New value = Old × (100±x)/100 | [02-Percentage-Change.md](./02-Percentage-Change.md) |
| Two successive % changes | Net% = a+b+ab/100 (use negative for decrease) | [02-Percentage-Change.md](./02-Percentage-Change.md) |
| % more/% less comparisons | Base matters (compare against reference value) | [03-Percentage-Applications.md](./03-Percentage-Applications.md) |

## Quick Values

- 1/2=50%, 1/3=33.33%, 1/4=25%, 1/5=20%, 1/8=12.5%
