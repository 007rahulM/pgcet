# Math 60-Second Playbook (Beginner Friendly)

This sheet is for students who feel math is hard under time pressure.

Use this exact order in exam:

1. **Recognize pattern** (Quadratic / Coordinate / Trigonometry / SD / DI / Figure counting)
2. **Pick one formula first** (do not try many formulas at once)
3. **Substitute fast**
4. **Option-check**
5. If stuck in **30 seconds**, mark and move

---

## 1) Pattern-First Decision Map

- **Quadratic** → convert to `ax²+bx+c=0` → check `D=b²-4ac` first.
- **Coordinate geometry** → difference in x/y first → slope/distance/line formula.
- **Trigonometry** → draw a small right triangle first → mostly `tan = opposite/adjacent`.
- **Statistics (SD/Variance)** → use property first (add constant, multiply constant) before full calculation.
- **DI (Pie/Line chart)** → read question first, then data.
- **Image/Figure counting** → count by levels (small → medium → large), never random counting.

---

## 2) Memory Hooks (Quick Recall)

- **Quadratic nature of roots:**  
  `D > 0` two real, `D = 0` equal, `D < 0` no real.
- **Line questions:** slope first, then line equation.
- **Heights & distances:** draw triangle first, then tan.
- **SD shortcut:**  
  add constant → SD same,  
  multiply by k → SD × |k|.
- **Pie chart angle:** `% × 3.6`.

---

## 3) 60-Second Rule (Very Important)

- **0–20 sec:** identify pattern + formula.
- **20–45 sec:** substitute and simplify.
- **45–60 sec:** option match and unit check.
- Not done by 60 sec → **skip and return**.

---

## 4) Diagram-First Topics (Draw Mini Sketch)

- Coordinate geometry (points/lines)
- Trigonometry (heights, depression/elevation)
- DI graphs (trend comparison)
- Counting figures / image-difference style visual questions

If printed diagram is unclear, draw a rough labeled copy and solve from that.

## 6) Don’t-Do List (Saves Marks)

- Don’t jump between formulas repeatedly.
- Don’t solve without pattern recognition.
- Don’t spend >60 sec on one hard question in first pass.
- Don’t avoid diagrams in diagram-based questions.
