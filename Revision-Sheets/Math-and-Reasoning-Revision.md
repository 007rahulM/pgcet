## рҹ§  REASONING

### Blood Relations Shortcuts
- Father/mother's sibling = Uncle/Aunt
- Sibling's child = Nephew/Niece
- Parent's parent = Grandparent
- Spouse's parent = Father/Mother-in-law
- **Draw tree, don't memorize steps!**

### Coding-Decoding
- Check shift: ABCDEвҶ’BCDEF = +1 each letter
- Reverse: AвҶ”Z(26), BвҶ”Y(25)... sum=27
- If first letter codes to specific, find pattern consistently

### Direction Sense
- Always draw: NвҶ‘ EвҶ’ SвҶ“ WвҶҗ
- After N then S of same distance = back to start
- Straight line distance = Pythagoras (aВІ+bВІ=cВІ)

### Series (Patterns)
- Arithmetic: +k each time
- Geometric: Г—k each time
- Squares: 1,4,9,16,25,36...
- Cubes: 1,8,27,64,125...
- Fibonacci: each = sum of previous 2

### Syllogisms (Quick Rules)
- All AвҶ’B, All BвҶ’C вҲҙ All AвҶ’C вң…
- All AвҶ’B, Some BвҶ’C вҲҙ Cannot conclude AвҶ’C вқҢ
- "Some" breaks the chain in syllogisms!

### Seating Arrangement
- Linear: work from definite clues first
- Circular: fix 1 person, arrange rest
- "Facing centre" = standard; "facing outside" = mirror (L/R reversed!)
