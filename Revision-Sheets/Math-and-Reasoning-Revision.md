# 🧮 Math & Reasoning — Quick Revision Sheet

## ⏱️ Time & Work
```
Together (2 people) = XY/(X+Y) days
LCM Method:
  1. Total work = LCM of all times
  2. Efficiency = LCM ÷ individual days
  3. Time = Total ÷ Combined efficiency
```
- Pipes: inlet (+), outlet (−), same LCM method

## 🚗 Time, Speed & Distance
```
D = S × T
Avg speed (equal dist) = 2ab/(a+b)
Relative speed: same dir = |a−b|; opp dir = a+b
Train crossing pole = Train length / speed
Train crossing platform = (Train + Platform) / speed
Downstream = B+S; Upstream = B−S
Boat = (D+U)/2; Stream = (D−U)/2
```
**Unit conversion:** km/h → m/s: × 5/18; m/s → km/h: × 18/5

## 💰 Profit & Loss
```
Profit = SP − CP
Profit% = (SP−CP)/CP × 100
SP = CP × (100+P%)/100
CP = SP × 100/(100+P%)
MP × (100−D%)/100 = SP
Successive: D1+D2−D1×D2/100
False weight profit% = Error/(True−Error) × 100
```

## 📊 Percentages
```
x% of y = xy/100
Successive change (a then b) = a+b+ab/100
```
**Fraction shortcuts:** 10%=1/10; 12.5%=1/8; 20%=1/5; 25%=1/4; 33⅓%=1/3; 50%=1/2; 66⅔%=2/3; 75%=3/4

## 🏦 Interest
```
SI = PRT/100
CI = P(1+R/100)^T − P  ←  or: A = P(1+R/100)^T
Difference (CI−SI) for 2 years = P(R/100)²
Half-yearly: use R/2 and 2T
```

## ⚖️ Averages
```
Avg = Sum ÷ Count
New number added: change in avg × new count = difference
```

## 🔢 Ratio & Proportion
```
a:b = c:d → ad = bc
Make middle term equal when combining ratios
Partnership: Profit ∝ Capital × Time
```

## 🔤 Permutation & Combination
```
nPr = n!/(n−r)!           [ORDER matters]
nCr = n!/[r!(n−r)!]        [order doesn't matter]
n! = n×(n−1)×...×1
0! = 1; 1! = 1
Circular = (n−1)!
Identical items = n!/repetitions!
```
**Common values:** 4P2=12; 5C2=10; 6C3=20; 5C3=10

## 🎲 Probability
```
P(E) = Favorable / Total
P(A or B) = P(A)+P(B)−P(A∩B)
P(A and B) = P(A)×P(B)  [independent events]
P(at least one) = 1 − P(none)
```
**Standard totals:** Coin×1=2; Coin×2=4; Die×1=6; Die×2=36; Cards=52 (4 suits×13)

## 🔵 Sets
```
n(A∪B) = n(A)+n(B)−n(A∩B)
n(A∪B∪C) = A+B+C − AB − BC − AC + ABC
Only A = n(A) − n(A∩B)   [for 2 sets]
2ⁿ subsets for n-element set
```

## 📈 Statistics
```
Mean = Σx/n
Median = middle value (sort first!)
Mode = most frequent
Range = Max − Min
Variance = Σ(x−mean)²/n
SD = √Variance
```
**Key property:** Adding constant to all → mean changes, SD unchanged

## 🔲 Matrices
```
2×2 det = ad−bc
A⁻¹ = (1/det) × adj(A)   [only if det≠0]
A×I = A (identity property)
|AB| = |A|×|B|
```

## 🔮 Logic
```
Truth values: T=1, F=0
∧ (AND): T only when both T
∨ (OR): F only when both F
→ (IF-THEN): F only when p=T, q=F
↔ (IFF): T when both same
De Morgan: ¬(A∧B)=¬A∨¬B; ¬(A∨B)=¬A∧¬B
p→q ≡ ¬q→¬p (contrapositive, EQUIVALENT)
p∨¬p = Tautology (always T)
p∧¬p = Contradiction (always F)
```

## 🧠 REASONING

### Blood Relations Shortcuts
- Father/mother's sibling = Uncle/Aunt
- Sibling's child = Nephew/Niece
- Parent's parent = Grandparent
- Spouse's parent = Father/Mother-in-law
- **Draw tree, don't memorize steps!**

### Coding-Decoding
- Check shift: ABCDE→BCDEF = +1 each letter
- Reverse: A↔Z(26), B↔Y(25)... sum=27
- If first letter codes to specific, find pattern consistently

### Direction Sense
- Always draw: N↑ E→ S↓ W←
- After N then S of same distance = back to start
- Straight line distance = Pythagoras (a²+b²=c²)

### Series (Patterns)
- Arithmetic: +k each time
- Geometric: ×k each time
- Squares: 1,4,9,16,25,36...
- Cubes: 1,8,27,64,125...
- Fibonacci: each = sum of previous 2

### Syllogisms (Quick Rules)
- All A→B, All B→C ∴ All A→C ✅
- All A→B, Some B→C ∴ Cannot conclude A→C ❌
- "Some" breaks the chain in syllogisms!

### Seating Arrangement
- Linear: work from definite clues first
- Circular: fix 1 person, arrange rest
- "Facing centre" = standard; "facing outside" = mirror (L/R reversed!)
