## 🎯 MASTER FORMULA SHEET

### 1. STANDARD FORM
```
ax² + bx + c = 0,  a ≠ 0
```

### 2. QUADRATIC FORMULA
```
x = (-b ± √(b² - 4ac)) / (2a)
```

### 3. DISCRIMINANT (D)
```
D = b² - 4ac
```

| D Value | Nature of Roots |
|--------|------------------|
| D > 0 | Two real & distinct |
| D = 0 | Real & equal |
| D < 0 | Imaginary (no real root) |

### 4. SUM & PRODUCT OF ROOTS
If roots are **α, β**:
```
α + β = -b/a
αβ = c/a
```

### 5. EQUATION FROM GIVEN ROOTS
If sum = S and product = P:
```
x² - Sx + P = 0
```

### 6. VERTEX FORM (COMPLETING SQUARE)
```
ax² + bx + c = a(x + b/2a)² - D/(4a)
```

## ⚡ 60-SECOND SHORTCUTS

- If **a + b + c = 0**, then **x = 1** is a root.  
- If **a - b + c = 0**, then **x = -1** is a root.  
- For integer roots, check factor pairs of **a × c**.  
- Use **D** first → saves time on impossible questions.

## 🔥 SOLVED EXAMPLES (EXAM STYLE)

### Example 1: Factorization
**Solve:** x² - 5x + 6 = 0  
```
x² - 5x + 6 = (x - 2)(x - 3) = 0
Roots: x = 2, 3
```

---

### Example 2: Quadratic Formula
**Solve:** 2x² + 3x - 2 = 0  
```
a=2, b=3, c=-2
D = 9 + 16 = 25
x = (-3 ± 5)/4
Roots: x = 1/2, -2
```

---

### Example 3: Nature of Roots
**Find nature of roots:** x² + 4x + 5 = 0  
```
D = 4² - 4(1)(5) = 16 - 20 = -4 < 0
No real roots
```

---

### Example 4: Equation from Roots
**Roots are 3 and -2. Find equation.**  
```
Sum S = 1, Product P = -6
Equation: x² - Sx + P = 0
x² - x - 6 = 0
```
