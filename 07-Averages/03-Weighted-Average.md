# Averages — Weighted Average & Combined Groups

## 🔍 When This Comes Up

- "Two groups combined — what is the average?"
- Different weights assigned to each value
- Mixing problems using averages

---

## 📐 Core Formula

### Weighted Average:
> **Weighted Avg = (w₁x₁ + w₂x₂ + ...) / (w₁ + w₂ + ...)**

Where w = weight (count/quantity), x = value

---

## ✅ Step-by-Step Examples

### Example 1 (Two Groups Combined)

**Problem:** Group A (20 students, avg 45), Group B (30 students, avg 55). Combined average?

- Weighted avg = (20×45 + 30×55) / 50
- = (900 + 1650)/50 = 2550/50 = **51** ✅

---

### Example 2 (Average Salary)

**Problem:** Average salary of 10 workers = ₹5000. Manager's salary = ₹15000. Average salary of all?

- 10 workers sum = 50000
- Total = 50000 + 15000 = 65000
- n = 11
- Avg = 65000/11 ≈ **₹5909** ✅

---

### Example 3 (Finding a Value from Two Averages)

**Problem:** Average of A, B = 30. Average of B, C = 25. Average of A, C = 35. Find A, B, C.

- A+B=60, B+C=50, A+C=70
- Adding: 2(A+B+C) = 180 → A+B+C = 90
- A = 90−50 = **40**, B = 90−70 = **20**, C = 90−60 = **30** ✅

---

### Example 4 (Mixture and Average)

**Problem:** A container has 30 litres of liquid with average concentration 40%. Another container has 20 litres at 60%. Combined concentration?

- Weighted avg = (30×40 + 20×60) / (30+20)
- = (1200 + 1200)/50 = 2400/50 = **48%** ✅

---

### Example 5 (Three Exam Averages)

**Problem:** Average of math = 70, science = 80, English = 65. Equal weightage. Combined average?

- = (70 + 80 + 65) / 3 = 215/3 ≈ **71.67** ✅

---

## ⚡ 60-Second Shortcut

**For two groups of equal size:** Combined avg = (avg1 + avg2) / 2

**For two groups of unequal size:** Use alligation:
```
Avg1                Avg2
    \              /
     \            /
   Combined avg
    /              \
(Avg2-Combined) : (Combined-Avg1)
```
This gives the ratio of group sizes.

## 📝 Practice Problems

1. Group X has 40 students with avg marks 50. Group Y has 60 students with avg marks 70. Combined average?

2. Average of 5 numbers = 30. Average of another 10 numbers = 15. Combined average of all 15?

3. A scored 75, 80, 65, 90 in 4 subjects. What must he score in 5th subject to have avg 78?

4. Average of A and B = 45. Average of C and D = 55. Average of A, B, C, D?

---

## ✔️ Answers

1. (40×50 + 60×70)/(40+60) = (2000+4200)/100 = **62**
2. (5×30 + 10×15)/15 = (150+150)/15 = **20**
3. Need total = 78×5 = 390. Current sum = 310. 5th subject score = 390−310 = **80**
4. Sum of A+B = 90, C+D = 110. Combined = 200/4 = **50**
