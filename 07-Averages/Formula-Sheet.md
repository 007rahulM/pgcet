# Averages — Formula Sheet

This sheet covers **age, weight, marks/score, salary** type average questions.

| If question says... | Use this formula/rule | Go to examples |
|---|---|---|
| Average of values | Average = Sum/n | [01-Basic-Average.md](./01-Basic-Average.md) |
| Find missing number using average | Missing = (Average×n) − known sum | [01-Basic-Average.md](./01-Basic-Average.md) |
| Person joins/leaves; age/weight changes | New sum = New avg×new n; compare with old sum | [02-Changed-Average.md](./02-Changed-Average.md) |
| Replacement (one out, one in) | New person = old person + (change in avg×n) | [02-Changed-Average.md](./02-Changed-Average.md) |
| Two groups with different averages | Weighted avg = (n1a1+n2a2)/(n1+n2) | [03-Weighted-Average.md](./03-Weighted-Average.md) |
| Batsman/class score new average | Update total sum, then divide by new count | [02-Changed-Average.md](./02-Changed-Average.md) |

## Core Formulas

- Sum = Average × Number of items
- Consecutive numbers average = (first+last)/2
