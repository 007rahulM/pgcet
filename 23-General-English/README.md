# General English — Overview

General English tests grammar, vocabulary, and reading comprehension. Expect **10–15 questions** in MCA PGCET.

## Quick Tips

- **Articles:** Listen to the SOUND (not spelling) — "an hour" ✅, "an MBA" ✅
- **Prepositions:** Since + point in time; For + duration
- **Tenses:** "just/already/since/for" → Present Perfect; "yesterday/ago" → Simple Past
- **Subject-Verb:** Collective nouns (team, family) = singular verb

## Study Order

Pattern 1 (Grammar) → Pattern 2 (Vocabulary) → Pattern 3 (Comprehension/Errors).
Focus more time on Vocabulary as it has the most guaranteed marks.
